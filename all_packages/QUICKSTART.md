## 📖 使用步驟

### 方法一：直接打開（最簡單）
1. 直接雙擊 `index.html` 文件
2. 網站將在您的預設瀏覽器中打開
3. 開始探索小雨論壇！

### 方法二：使用本地服務器（推薦）
```bash
# 如果您安裝了 Python
python -m http.server 8000

# 如果您安裝了 Node.js
npx serve .

# 然後在瀏覽器訪問
http://localhost:8000
```

## ⚠️ 注意事項

### 數據持久化
- 所有數據保存在瀏覽器的 LocalStorage
- 清除瀏覽器數據會導致信息丟失
- 不同瀏覽器的數據是獨立的

### 模擬功能
以下功能是模擬實現（演示版本）：
- 交易結算（無真實區塊鏈交互）
- 設備驗證（無實際多設備驗證）
- 用戶互動（無實時通訊）

## 🆘 常見問題

### Q: 註冊後找不到我的帳號？
A: 檢查是否使用了相同的瀏覽器。數據保存在 LocalStorage 中。

### Q: 如何重置所有數據？
A: 打開開發者控制台，執行：
```javascript
localStorage.clear();
location.reload();
```

### Q: XYC餘額不足怎麼辦？
A: 前往公佈欄領取任務獎勵，或重新註冊獲得獎勵。

### Q: 手機上顯示不正常？
A: 確保使用現代瀏覽器，並清除瀏覽器緩存。
