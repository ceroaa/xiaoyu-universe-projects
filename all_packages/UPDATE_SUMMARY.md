## 📦 提交作品要求

### 必須提供的內容

1. **算力使用證明**
   - GPU使用時間記錄
   - 性能監控截圖
   - 使用日誌文件

2. **任務完成證明**
   - 模型訓練結果（如適用）
   - 處理完成的文件
   - 計算輸出結果

3. **作品連結**
   - 雲端硬碟連結（Google Drive、OneDrive等）
   - GitHub倉庫連結
   - 臨時文件分享連結

### 提交示例
```
作品說明：
已使用NVIDIA A100提供24小時算力，完成GPT模型訓練。
訓練過程穩定無中斷，GPU利用率保持在95%以上。

GPU使用記錄：
- 開始時間：2026-03-08 10:00
- 結束時間：2026-03-09 10:30
- 總時長：24.5小時
- 平均利用率：96%

提交內容：
1. 訓練完成的模型：https://drive.google.com/model.pth
2. GPU監控截圖：https://imgur.com/gpu-monitor
3. 訓練日誌：https://pastebin.com/training-log

作品連結：
https://drive.google.com/folder/abc123
```
