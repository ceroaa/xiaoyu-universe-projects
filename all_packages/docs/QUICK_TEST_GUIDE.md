## 🔍 故障排除

### 問題 1: 看不到其他用戶的留言

#### 可能原因：
1. 新 JS 文件未載入
2. 瀏覽器快取

#### 解決方法：
```
1. 清除瀏覽器快取：Ctrl + Shift + Delete
2. 強制重新整理：Ctrl + F5 (Windows) 或 Cmd + Shift + R (Mac)
3. 檢查 Console 是否有錯誤訊息
```

#### 檢查 JS 載入：
```javascript
// 打開瀏覽器 Console，執行：
console.log(typeof window.userManager);      // 應返回 "object"
console.log(typeof window.moodPostsManager); // 應返回 "object"
```

---

### 問題 2: 註冊後沒有自動登入

#### 檢查步驟：
```javascript
// 打開瀏覽器 Console，執行：
console.log(localStorage.getItem('isLoggedIn')); // 應返回 "true"
console.log(localStorage.getItem('currentUser')); // 應顯示用戶資料
```

#### 解決方法：
- 手動重新整理頁面（F5）
- 清除 localStorage 後重新註冊：
  ```javascript
  localStorage.clear();
  location.reload();
  ```

---

### 問題 3: 發布留言後沒有顯示

#### 檢查步驟：
1. 打開瀏覽器 Console
2. 查看是否有紅色錯誤訊息
3. 檢查 Network 標籤，查看 API 請求狀態

#### 解決方法：
```javascript
// 手動測試 API
fetch('tables/forum_posts?limit=10')
  .then(res => res.json())
  .then(data => console.log('API 正常:', data))
  .catch(err => console.error('API 錯誤:', err));
```

---

### 問題 4: 舊版 localStorage 資料衝突

#### 症狀：
- 登入後顯示舊的用戶資料
- 留言顯示不正確

#### 解決方法：
```javascript
// 清除所有舊資料
localStorage.clear();
sessionStorage.clear();
location.reload();

// 然後重新註冊
```
