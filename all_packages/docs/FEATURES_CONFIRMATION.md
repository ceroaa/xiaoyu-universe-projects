## 🎨 關於 404 頁面的說明

### 為什麼沒有 404？

**設計決策**：單頁應用通常不需要傳統的 404 頁面：

```
傳統網站：
/nonexistent → 404 Not Found

單頁應用：
/nonexistent → 返回 index.html → JavaScript 處理路由
              → 可以顯示「頁面不存在」或重定向到首頁
```

### 如果需要 404 頁面

可以在前端 JavaScript 中添加路由檢查：

```javascript
// 檢查當前路徑是否有效
const validPaths = [
  '/',
  '/index.html',
  '/profile.html',
  '/ai-chat.html',
  '/exchange.html',
  '/data-manager.html'
];

if (!validPaths.includes(window.location.pathname)) {
  // 顯示 404 頁面或重定向
  window.location.href = '/index.html';
}
```
