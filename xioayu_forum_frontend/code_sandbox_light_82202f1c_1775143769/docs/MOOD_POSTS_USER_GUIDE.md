# 💭 心情動態系統 - 使用指南

## 🎯 功能概述

心情動態系統是小雨論壇個人中心的全新功能，讓用戶可以記錄和分享日常心情、想法和感受。

---

## 📍 如何使用

### 1. 進入心情動態
1. 登入小雨論壇
2. 點擊導航欄「**個人中心**」
3. 切換到「**最近活動**」標籤
4. 看到心情發布區域

### 2. 發布心情動態

#### 方式一：點擊發布
```
1. 在文字框輸入內容（最多 500 字）
2. 觀察右下角字數統計
3. 點擊「發布動態」按鈕
4. 看到成功提示 ✨
```

#### 方式二：快捷鍵
```
1. 在文字框輸入內容
2. 按下 Ctrl + Enter
3. 自動發布
```

### 3. 管理動態

#### 查看動態
- 所有動態按時間倒序排列
- 顯示發布時間（智能格式化）
- 即時字數統計

#### 按讚動態
- 點擊 ❤️ 圖標
- 數字立即增加
- 可多次按讚

#### 刪除動態
- 點擊「🗑️ 刪除」按鈕
- 確認刪除提示
- 動態永久移除

## ⚙️ 功能特性

### 字數統計
- **正常**（0-400字）：灰色字體
- **警告**（401-450字）：橙色字體 ⚠️
- **危險**（451-500字）：紅色字體 🔴
- **超出**（>500字）：無法發布 ❌

### 時間顯示
系統會自動格式化時間：
- 剛剛（< 1 分鐘）
- X 分鐘前（< 1 小時）
- X 小時前（< 1 天）
- X 天前（< 1 週）
- X 週前（< 1 個月）
- X 個月前（≥ 1 個月）

### 數據存儲
- 💾 使用 localStorage 本地存儲
- 🔄 自動保存，無需手動操作
- 🔒 僅存儲在本地瀏覽器
- 📊 支持無限條動態（受瀏覽器限制）

## 🎯 使用技巧

### 1. 快速發布
使用 `Ctrl + Enter` 快捷鍵可快速發布，無需點擊按鈕。

### 2. 清空內容
點擊「清空」按鈕可快速清除輸入框內容，重新開始。

### 3. 字數控制
- 輸入時觀察字數變化
- 超過 400 字會變色提醒
- 建議控制在 300 字以內，表達更簡潔

### 4. 動態排序
新發布的動態會自動置頂，方便查看最新內容。

### 數據安全
- ✅ **本地存儲**：數據僅存在您的瀏覽器

## ❓ 常見問題

### Q1: 我的動態為什麼消失了？
**A**: 可能原因：
1. 清除了瀏覽器緩存/Cookie
2. 使用了無痕模式
3. 更換了設備/瀏覽器

**解決方案**：動態存儲在本地，清除數據會丟失。

### Q2: 可以插入圖片嗎？
**A**: 目前僅支持純文字動態，圖片功能在規劃中。

### Q3: 別人能看到我的動態嗎？
**A**: 不能。動態僅存儲在您的本地瀏覽器，完全私密。

### Q4: 動態數量有限制嗎？
**A**: 沒有固定限制，但受瀏覽器 localStorage 容量限制（通常 5-10MB）。

### Q5: 如何備份動態？
**A**: 目前需要手動複製文字。未來會添加導出功能。

### Q6: 刪除的動態可以恢復嗎？
**A**: 無法恢復。刪除前會有確認提示，請謹慎操作。
