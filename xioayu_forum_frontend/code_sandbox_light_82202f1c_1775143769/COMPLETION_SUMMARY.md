## 🚀 如何使用

### 方式一：本地運行

```bash
# 直接打開
open index.html

# 或使用 HTTP 伺服器
python -m http.server 8000
# 訪問 http://localhost:8000
```

### 方式二：在線發布

1. 前往 **Publish 標籤頁**
2. 點擊 **發布** 按鈕
3. 獲取公開 URL
